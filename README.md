# Libeldata — Landing Page

## Estructura del proyecto
```
libeldata/
├── index.html        → Estructura HTML de la página
├── css/
│   └── styles.css    → Todos los estilos
├── js/
│   └── main.js       → Animaciones, FAQ, carrusel, hero dinámico
├── img/              → Carpeta para imágenes propias (vacía por ahora)
└── README.md         → Este archivo
```

## Cómo subir a Netlify
1. Arrastra la carpeta `libeldata/` completa a netlify.com/drop
2. Listo — tu página queda publicada en segundos

## Pendiente antes de publicar
- Reemplaza `57TUNUMERO` con tu número real de WhatsApp en index.html
- Reemplaza `hola@libeldata.com` con tu email real
- Compra el dominio libeldata.com en Namecheap y conéctalo en Netlify

## Notas
- Las imágenes de fondo vienen de Unsplash (URL externa, no necesitan descargarse)
- Los logos del carrusel vienen de CDNs públicos
